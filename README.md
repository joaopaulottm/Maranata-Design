## Mas o que é um framework?

Framework é um conjunto de bibliotecas (biblioteca é um conjunto de classes ou metodo) para facilitar o 
desenvolvimento de site e sistemas. Os framework mais conhecidos de css são o Bootstrap do Twitter e o 
Material Design do Google mas existe muitos outros.

## Com esse meu framework será mais facil de você aprender

Com esse meu framework será mais facil de você aprender HTML5 e CSS3 porque é em portugues
e eu explico com detalhes esse meu framework que esta em processo de desenvolvimento.

Ele é extencivel, ou seja ele é dividido em varios arquivos cada arquivo responsavel por uma estilização especifica.
Aí você adiciona os arquivos que você vai precissar e assim não precissarár carregar classes sem necessidades com isso o
seu site ou sistema fica mais rapido de carregar.
