# Maranata Design

Maranata Design é um framework Css escrito na lingua portuguesa, no portugues do Brasil.
Esse Framework é 100% Brasileiro.
Espero que você use com sabedoria.

Meu nome é João Paulo Sena Padilha e sou um Programador Web minha formação é de um curso tecnico de informatica
do IFRN (Institudo Federal do Rio Grande do Norte)

## Com esse meu framework será mais facil de você aprender

Com esse meu framework será mais facil de você aprender HTML5 e CSS3 porque é em portugues
e eu explico com detalhes esse meu framework que esta em processo de desenvolvimento.

Ele é extencivel ele e dividido em varios arquivos cada arquivo responsavel por uma estilização especifica.
Aí você os arquivos que você precissar e assim não precessarár carregar classes sem necessidades com isso o
seu site ou sistema fica mais rapido de carregar.
